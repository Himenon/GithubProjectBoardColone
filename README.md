# Clone Github Dashboard

## Prepare

- graphql-cli: <https://github.com/graphql-cli/graphql-cli>

```bash
npm install -g graphql-cli graphql-cli-load
```

## Connect Check

```bash
graphql ping --dotenv .env
```

## Usage

### Get Schema

```bash
graphql get-schema -p clone-github-project-board --dotenv .env
```
